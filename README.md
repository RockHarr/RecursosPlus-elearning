

# RecursosPlus-elearning – Trabajo colaborativo

Este repositorio corresponde a un **trabajo colaborativo** del curso de Frontend.  
La tarea consiste en implementar una página web en grupos, que incluya las secciones:

- Quiénes somos  
- Misión  
- Visión  
- Nuestros productos o servicios  

y dejar el proyecto respaldado en GitHub.

---

## Objetivo del proyecto

- Practicar la creación de una página web básica en equipo.
- Aplicar estructura semántica en HTML (`header`, `main`, `section`, `aside`, `footer`).
- Definir y escribir contenidos simples: quiénes somos, misión, visión, servicios.
- Utilizar GitHub como herramienta de trabajo colaborativo:
  - subir el proyecto,
  - compartirlo entre los integrantes,
  - dejar registro de los cambios.
